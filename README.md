This repository hosts the CSS and JavaScript assets used by the  
**Upload Memories** page on the Eonic Bonds Webflow site.

These files are loaded via jsDelivr CDN, allowing the Webflow site  
to update automatically whenever new commits are pushed.

## CDN URLs

### CSS
- Base styles  
  https://cdn.jsdelivr.net/gh/eonicbonds/eonic-bonds-memories-assets@main/css/base.css

- Memory page styles  
  https://cdn.jsdelivr.net/gh/eonicbonds/eonic-bonds-memories-assets@main/css/memories.css

### JS
- Core helpers  
  https://cdn.jsdelivr.net/gh/eonicbonds/eonic-bonds-memories-assets@main/js/memories-core.js

- Page logic  
  https://cdn.jsdelivr.net/gh/eonicbonds/eonic-bonds-memories-assets@main/js/memories-page.js

## Deployment

1. Push commits to `main`
2. jsDelivr updates automatically
3. Webflow reflects the changes instantly (no republishing needed)
